开发技术、组件开发、组件通信

一、开发技术

项目构建

  • Vite：支持本地快速冷启动、热更新和按需编译的非打包服务器，基于 esbuild 预构建依赖并在浏览器请求模块时才进行真正的按需编译，但是打包到生产环境时需要通过 rollup。
  • Webpack：启动后从入口开始逐步进行语法解析、依赖收集、代码转译、打包合并、代码优化等一系列操作，最终将高版本的、离散的源码编译打包成低版本、高兼容的代码。

版本优化

废弃了过滤器功能、$on/$emit/$off/$once 等实例方法、支持多根节点组件和 TypeScript、使用函数的 Composition API (响应式 ref/生命周期 onMounted /依赖注入 provide 等一系列接口的集合) 取代了声明变量的选项式 API、通过 Proxy + Reflect 取代 Object.definepeoperty() 实现的响应式对象可以直接操作而不需要每次遍历对象属性。

$attrs
