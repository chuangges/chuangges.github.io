CSS 常用、PC 样式、Mobile 样式、Scss 预处理

一、CSS 常用

浏览器兼容

  • img 之间出现间距：浮动。
  • 上下 margin 重合问题：上下选一个。
  • li 元素使用 display: inline-block 时会出现缝隙：浮动、font-size: 0。
  • opacity、placeholder、outline、伪元素：兼容方案 CSS Hack (前缀、IE 注释)。
  • 子元素 margin-top 出现在父元素上方：父元素 overflow: hidden / padding-top。

css 单位

  • rem: 相对于根元素字体尺寸
  • em: 相对于父级元素的字体尺寸
  • %: 百分比，相对于父元素的尺寸
  • px: 像素，相对于用户设备分辨率的最基本单元
  • pt：Word等办公软件中使用的字体大小单位
  • dp：安卓开发时使用的长度单位
  • sp：安卓开发时使用的字体大小单位
  • vw：视窗宽度，1vw等于视窗宽度的 1%
  • vh：视窗高度，1vh等于视窗高度的 1%
  • vmin、vmax：vw 和 vh 中较 小/大 的那个
  • dvh：内容高度，会自动忽略浏览器导航栏高度。
  • lvh：最大高度，包含浏览器导航栏高度。
  • svh：最小高度，不包含浏览器导航栏高度。

transform 变换

  • 变换属性：transform-origin 中心点、transform-style 场景、perspective 视距、perspective-origin 视距基点、backface-visibility 是否可看场景背面。
  • 2D 变换：translate: translate[X/Y] 位移、scale: scale[X/Y]、rotate: rotate 旋转、skew: skew[X/Y] 变形、matrix 矩阵(2D 变换的本质)。
  • 3D 变换：translate3d: translate(3d/Z) 位移、scale3d: scale(3d/Z) 缩放、rotate3d: rotate(3d/X/Y/Z) 旋转、matrix3d 矩阵(3D 变换的本质)。

transition 过渡

默认样式中定义元素的初始样式和过渡函数，然后定义过渡后的元素样式。

animation 动画

通过 animation 绑定 @keyframes 定义的动画，自动实现一种或多种形态同时变化的效果，精彩案例可点击查看。其它实现方式有：setTimeout (指定间隔更新样式但导致频繁重绘)、requestAnimationFrame (根据浏览器的刷新频率实现最佳绘制)、Animate.css：(只需为元素添加类名的动画库)、Tween.js (各种经典算法的补间动画可实现平滑效果)、Three.js：(基于浏览器接口 webgl 实现 3D 动画)、swiper.animate.js (移动端触摸滑块类库)。

组件样式

  • Scoped：在元素中添加了一个唯一属性，注意样式修改时必须定位属性值。
  • Module：通过插件 css-loader 等配置的命名规则将 :class 绑定的类名重新生成。

布局方式

瀑布流布局

原理是页面容器内的多个高度不固定的 div 之间按照一定的间隔参差不齐的无序浮动，鼠标滚动时不断在容器内的尾部加载数据并自动加载到空缺位置，不断循环。
核心是基于一个网格的布局，而且每行包含的项目列表高度是随机的（随自身内容动态改变高度），同时每个项目列表呈堆栈形式排列，最关键的是，堆栈之间彼此之间没有多余的间距差存大。
