操作 html 的 DOM、与浏览器交互的 BOM

一、DOM

文档对象模型，是一种独立于语言并用于操作 xml、html 文档的应用编程接口。它可以将任何 xml / html 文档描绘成一个由多层节点构成的树状结构，并允许 Javascript 进行增删改查等操作。

DOM 树

html 标签通过浏览器的解析后才会形成 dom 树，其中的每个 标签元素、属性、文本 都可以看作是一个 DOM 节点，JavaScript 可以通过 dom 的提供的编程接口操作到每个节点。



元素节点

文本节点、属性节点都属于元素节点的子节点，操作时都需要先获取 元素节点

  1. 创建：document.createElement
  2. 获取
    • document.getElementBy Id/TagName/ClassName/Name
    • document.querySelector [All]
    • document.documentElement
    • document.body
    • document.forms

属性节点

  1. 查看：getAttribute
  2. 设置：setAttribute

文本节点

  1. 设置：innerText、innerHTML
